# 📋 Basecamp Board — Team Project Tracker

A simple Kanban-style project tracker built with plain HTML, CSS, and JavaScript.
Track projects across **To Do → In Progress → Done**, with deadlines and team members.

Deadline chips automatically change color:
- 🟢 Green — more than 7 days left
- 🟠 Amber — 7 days or less
- 🔴 Red — overdue

Data is saved in your browser's local storage, so it persists between visits.

## Features
- Add new projects with a name, deadline, and team members  
- Move projects between columns
- Color-coded deadline urgency
- No backend or install required — just open `index.html`

## Tech
- HTML5
- CSS3 (custom properties, grid, flexbox)
- Vanilla JavaScript (no frameworks)

## Running locally
Just open `index.html` in your browser. No build step needed.
